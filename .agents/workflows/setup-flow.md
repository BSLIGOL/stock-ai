# 🛠️ 개발 환경 구축 워크플로우

이 워크플로우는 Docker DB 컨테이너 실행 및 스키마 초기화를 자동화/정형화합니다.

## 1. 인프라 확인
- [ ] Docker 가 실행 중인지 확인합니다.
- [ ] `docker-compose.yml` 설정(포트 5432 등)을 확인합니다.

## 2. DB 컨테이너 실행
// turbo
- [ ] `cmd /c docker-compose up -d` 명령어를 통해 전용 DB 컨테이너를 실행합니다.

## 3. 스키마 초기화
- [ ] `schema_setup.sql` 파일을 실행하여 테이블 및 pgvector 확장을 설정합니다.
- [ ] 초기 마스터 데이터(`stocks` 등)가 정상적으로 구성되었는지 확인합니다.

## 4. 어플리케이션 실행 확인
// turbo
- [ ] `./gradlew bootRun` 명령어로 어플리케이션이 정상적으로 구동되는지 확인합니다.
